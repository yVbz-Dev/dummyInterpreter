# eLang 🗺️

eLang is a personal project aimed at creating a robust interpreted language. Developed with Go, it currently supports:

 - Variable Assignments for dynamic data storage.

 - Standard Input/Output via print statements.

 - Control Flow with functional if/else logic.

 - Arithmetic Operations for core mathematical evaluations.

## Syntax Example 💻

```
var user = "Vbz"
var status = "active"

if (status == "active") {
    print("Welcome to eLang, " + user)
} else {
    print("System offline")
}
```


Output

```
Welcome to eLang, Vbz
```
